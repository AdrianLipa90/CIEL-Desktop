## Why this folder exists

The desktop repository should not only contain launchers and interface layers.
It should also expose the formal specification surfaces that explain what kind of truth contract the UI is expected to respect.

## Semantic role

This folder is for protocol-level documents, not for implementation runbooks.

- runbooks tell a human how to operate the system
- indices tell a human where things live
- specifications tell a human what the system is trying to preserve
